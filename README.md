# 朋友圈分析交友助手使用文档

## 项目介绍

朋友圈分析交友助手是一款基于AI技术的H5应用，旨在通过分析对方朋友圈截图，帮助用户更全面地了解心仪对象，从而提高交友成功率。

### 核心功能
- 批量上传朋友圈截图（支持拖拽上传）
- AI智能分析图片内容，提取个人信息
- 生成详细的个人分析报告
- 提供针对性的交友建议
- 保护用户隐私，分析完成后自动删除图片

### 技术栈
- **前端**：HTML5、CSS3、JavaScript
- **后端**：Node.js、Express
- **AI模型**：豆包大模型（Doubao-Seed-1.6-vision）

## 环境配置

### 前提条件
- Node.js 16.0 或更高版本
- npm 7.0 或更高版本
- 火山引擎方舟平台API Key

### 配置步骤

1. **克隆项目**（如果尚未克隆）
   ```bash
   git clone <项目地址>
   cd pyqfenxi
   ```

2. **安装后端依赖**
   ```bash
   cd backend
   npm install
   ```

3. **配置API Key**
   - 打开 `backend/.env` 文件
   - 将 `ARK_API_KEY` 替换为你的火山引擎方舟平台API Key
   - 确保 `ARK_MODEL` 设置为 `doubao-seed-1-6-vision-250815`

   ```env
   # 服务器配置
   PORT=3001

   # 豆包大模型API配置
   ARK_API_KEY=your_api_key_here
   ARK_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
   ARK_MODEL=doubao-seed-1-6-vision-250815
   ```

4. **启动后端服务器**
   ```bash
   npm start
   ```
   服务器将运行在 `http://localhost:3001`

5. **关闭后端服务器**
   - 在运行服务器的终端中，按下 `Ctrl+C` 组合键
   - 服务器将停止运行
   - 如需重新启动，再次执行 `npm start` 命令

## 使用步骤

### 1. 打开应用
- 在浏览器中打开 `frontend/index.html` 文件
- 你将看到应用的主界面

### 2. 上传朋友圈截图
- 点击上传区域或拖拽图片到上传区域
- 支持批量上传多张图片
- 至少需要上传5张图片才能进行分析
- 上传后可以预览和删除图片

### 3. 开始分析
- 确认上传了至少5张图片后，点击"开始分析"按钮
- 系统将显示分析进度条和剩余时间
- 后端服务器会调用豆包大模型API进行分析

### 4. 查看分析报告
- 分析完成后，系统将自动显示分析报告
- 报告包含以下内容：
  - **性格特点**：提取的性格标签
  - **兴趣爱好**：分析出的兴趣爱好及描述
  - **生活方式**：生活习惯和方式的分析
  - **社交偏好**：社交行为和偏好的分析
  - **交友建议**：包括话题推荐、约会建议、沟通技巧和关系进展建议

### 5. 分享或保存报告
- 可以点击"微信分享"或"微博分享"按钮分享报告
- 可以点击"保存报告"按钮保存分析结果
- 点击"重新分析"按钮可以上传新的图片进行分析

## 功能详解

### 图片上传
- **支持格式**：JPG、PNG、WEBP
- **大小限制**：每张图片不超过10MB
- **数量要求**：至少上传5张图片
- **操作方式**：点击上传或拖拽上传

### AI分析
- **模型**：使用豆包大模型Doubao-Seed-1.6-vision
- **分析维度**：性格特点、兴趣爱好、生活方式、社交偏好
- **分析原理**：通过多模态技术分析图片内容，提取关键信息
- **隐私保护**：分析完成后自动删除上传的图片

### 分析报告
- **性格特点**：以标签形式展示，如乐观开朗、热爱生活等
- **兴趣爱好**：详细列出兴趣点及描述
- **生活方式**：综合分析生活习惯和品质
- **社交偏好**：分析社交行为和人际关系
- **交友建议**：
  - **话题推荐**：基于兴趣爱好的聊天话题
  - **约会建议**：适合的约会场所和活动
  - **沟通技巧**：有效的沟通方式和技巧
  - **关系进展**：关系发展的策略和建议

## 常见问题

### 1. 为什么需要至少上传5张图片？
- 为了获得更全面、准确的分析结果，需要足够的样本数据
- 5张图片可以覆盖不同场景和时间的内容，提高分析的代表性

### 2. 上传的图片会被保存吗？
- 不会。系统仅在分析过程中临时存储图片，分析完成后会自动删除所有上传的图片
- 保护用户隐私是我们的首要原则

### 3. 分析结果准确吗？
- 分析结果基于AI模型对图片内容的理解，具有一定的参考价值
- 但由于朋友圈内容的多样性和复杂性，分析结果仅供参考，不能完全代表个人真实情况
- 建议结合实际接触和了解，综合判断对方的性格和特点

### 4. 为什么分析时间较长？
- 分析过程需要调用AI模型API，处理多张图片内容
- 网络延迟和模型处理时间会影响分析速度
- 一般情况下，分析过程需要15-30秒

### 5. 如何获取火山引擎方舟平台API Key？
- 访问火山引擎官网：https://www.volcengine.com/
- 登录账号并进入控制台
- 找到"方舟大模型服务平台"
- 进入"API Key 管理"页面
- 创建新的API Key
- 将生成的API Key复制到 `.env` 文件中
